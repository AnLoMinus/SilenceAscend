# 🌟 מהות תענית דיבור — שליטה, ניקוי ושדרוג של הנפש

תענית דיבור היא פעולה אנרגטית־רוחנית שמנקה את מערכת הדיבור, ממיינת את האגו, מרגיעה את המחשבה ומחדדת את הכלי הפנימי.
זו לא תענית של סבל — זו תענית של עוצמה.
מטרתה: **להפעיל מחדש את כוח המילה** וליישר אותו עם האמת, השליטה והכוונה של האדם.

תענית דיבור = **הפסקה מוחלטת מכל דיבור שאינו קדוש, הכרחי או מלא משמעות.**
במצב הזה—הדיבור נהיה **מכוון, נקי, חד ויוצר מציאות**.

# 🔥 למה עושים תענית דיבור?

## 🎯 מה מפעילה התענית בנפש?

### 1️⃣ **🧘‍♂️ ריסון היצר של הפה**

הפה הוא אחד השערים העוצמתיים ביותר.
שקט חיצוני → שקט פנימי → שליטה רגשית.

### 2️⃣ **🎤 ריפוי כוח הדיבור**

הדיבור חוזר להיות כלי של יצירה, ברכה והשפעה ולא של פיזור, לשון הרע או עצבנות.

### 3️⃣ **⚡ איפוס אנרגטי**

הורדת רעשים → פתיחת אינטואיציה → חידוד הקול הפנימי.

### 4️⃣ **🌌 חיזוק החיבור לבורא**

פחות מילים = יותר נוכחות, מודעות וקדושה.
